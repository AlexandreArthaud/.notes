# Linux

```aln
# Generic
  Command to copy the contents of {folder1} into {folder2} (preserving symlinks, file permissions, timestamps, etc) : rsync -a folder1/ folder2/

# Package Managers
  # Aptitude
    (Apt) Command to install a {package} : apt-get install package
    (Apt) Command to remove a {package} : apt-get remove package
    (Apt) Command to remove a {package}, along with any related configuration file : apt-get purge package
    (Apt) Command to update a {package} : apt-get remove package
    (Apt) Command to upgrade a {package} : apt-get upgrade package
```
