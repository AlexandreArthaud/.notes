# Latin

## Lexicon

```aln-duolingo
sum {sum-es-est-sumus-estis-sunt esse fuī futūrus} : to be
fēmina : woman ; wife
vir : man ; husband
puer : child ; boy / lad
puella : girl
ego / egō {meī mihi mē_} : I
salveō {salvēre} : to be well
  salvē! * salvēte! : hello
quis / quī {quid quae/qua} : who <in masculine form> ; what <in neuter form>
nōmen : name
  quid tibi est nomen? : what is your name?
  nomen mihi est X : my name is X
quōmodo : how / in what manner
  quōmodo te habes? : how are you doing?
  me bene habeō : I am well
habeō {2: habēre habuī habitum} : to have
bene {melius optimē} : well
dormiō {4: dormīre dormīvī/dormiī dormītum} : to sleep
nōn : not
sed : but
tū {tuī tibi tē_} : you
urbs {f3: urbis} : city <walled town>
is {id ea} : he / she / it
domus {f4/2: domūs/domī} : house
  domī {domibus} : at home
studeō {2: studēre studuī} : to strive after [DAT] ; to study #Late/Medieval
scrībō {3; scrībere scrīpsī scrīptum} : to write
et : and
māter {f3: mātris} : mother
pater {m3: patris} : father
frāter {m3: frātris} : brother
soror {f3: sorōris} : sister
```
